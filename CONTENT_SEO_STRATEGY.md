## üéØ SEO Optimization Checklist

### On-Page SEO (Per Blog Post)
- [ ] Target keyword in title (first 60 characters)
- [ ] Meta description with keyword (150-160 characters)
- [ ] H1 tag with primary keyword
- [ ] H2/H3 tags with semantic keywords
- [ ] Keyword density 1-2%
- [ ] Internal links to 3-5 relevant pages
- [ ] External links to 1-2 authoritative sources
- [ ] Image alt text with descriptive keywords
- [ ] URL slug with primary keyword
- [ ] Schema markup (Article, FAQ)

### Content Quality Standards
- **Minimum Length:** 1000 words (ideal: 1500-2500)
- **Readability:** Grade 8-10 level
- **Structure:** Short paragraphs (3-4 sentences)
- **Formatting:** Bullet points, numbered lists, tables
- **Media:** Minimum 2-3 images, 1 infographic
- **Originality:** 100% unique, no plagiarism
- **Value:** Actionable advice, local insights

### Local SEO Optimization
- [ ] Include "Ranchi" or "Jharkhand" in meta tags
- [ ] Add local landmarks in content
- [ ] Include contact information
- [ ] Embed Google Maps
- [ ] Local business schema markup
- [ ] Google My Business optimization
- [ ] Local backlinks from Ranchi directories
